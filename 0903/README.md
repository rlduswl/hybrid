# 📱 코르도바(Cordova)를 이용한 앱 개발 수업

## 1. 수업 개요
Apache Cordova는 HTML, CSS, JavaScript를 이용하여 **하이브리드 모바일 앱**을 개발할 수 있는 오픈소스 프레임워크이다.  
한 번의 코드 작성으로 iOS, Android 등 다양한 플랫폼에 배포할 수 있다는 장점이 있다.

- **대상**: 웹 기술을 활용하여 모바일 앱을 개발하고 싶은 학생
- **목표**: Cordova를 활용하여 기본적인 앱을 제작 및 배포할 수 있는 역량 습득

---

## 2. 학습 목표
- Cordova의 개념 및 특징 이해
- Cordova 개발 환경 구축
- Cordova 프로젝트 생성 및 실행
- 기본 플러그인 사용법 학습 (카메라, 파일, 네트워크 등)
- 하이브리드 앱의 장단점 이해
- 간단한 앱 기획 → 제작 → 테스트 → 배포 실습

---

## 3. 수업 커리큘럼

### 1주차: 오리엔테이션 & 개발 환경 구축
- 하이브리드 앱과 네이티브 앱 비교
- Cordova 개요 및 활용 사례
- Node.js, Cordova CLI 설치
- Android Studio / Xcode 환경 설정

### 2주차: Cordova 프로젝트 기본
- Cordova 프로젝트 생성 및 실행
- 프로젝트 구조 이해
- 웹 리소스(HTML, CSS, JS)와 Cordova 연동
- 디버깅 방법 학습

### 3주차: Cordova 기본 플러그인 활용
- Device Plugin: 기기 정보 가져오기
- Network Plugin: 네트워크 상태 확인
- Camera Plugin: 카메라로 사진 촬영 및 저장
- File Plugin: 파일 읽기/쓰기

### 4주차: UI 및 기능 확장
- 모바일 친화적인 UI 구현 (Ionic Framework, Onsen UI 등 연계)
- 이벤트 처리 및 사용자 입력 관리
- 로컬 스토리지 활용
- 외부 API 연동 (예: 날씨 API, 지도 API)

### 5주차: 앱 기획 및 팀 프로젝트
- 간단한 앱 아이디어 기획
- 와이어프레임 설계
- 역할 분담 후 개발 시작

### 6주차: 프로젝트 개발 및 테스트
- 기능 구현 및 디버깅
- Cordova 빌드(Android/iOS)
- 실제 기기 테스트

### 7주차: 앱 완성 & 발표
- 프로젝트 완성 및 실행 시연
- 배운 점 및 개선할 점 발표
- 앱 스토어 배포 방법 소개

---

## 4. 실습 예제
- **Hello Cordova**: 기기 정보 출력 앱
- **카메라 활용 앱**: 사진 촬영 후 화면에 표시
- **할 일 관리 앱(To-Do App)**: 로컬 스토리지 기반 간단한 CRUD 앱
- **API 연동 앱**: 외부 공공데이터/오픈 API 사용
